# Controle de Prazos

Projeto **Controle de Prazos** desenvolvido em Python para gerenciar e acompanhar prazos de processos ou tarefas de migração.

## Funcionalidade

- Controle de status de migração (adiantada, no limite, atrasada)
- Possibilidade de migração **com adequação** ou **sem adequação**
- Cálculo automático de dias entre datas de envio, adequação e etapas internas

## Como usar

1. Clone o repositório:

```bash
git clone https://github.com/SEU_USUARIO/controle_de_prazos.git
```

# Deadline Control

**Deadline Control** is a Python project developed to manage and track migration process or task deadlines.

## Features

- Tracks migration status (ahead of schedule, at the limit, overdue)
- Option for **migration with adjustment** or **migration without adjustment**
- Automatic calculation of days between submission, adjustment, and internal steps
